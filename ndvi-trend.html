The question

Is the Earth getting greener or browner?

Using NASA MODIS NDVI (Normalized Difference Vegetation Index) satellite data spanning 2001-2024, this visualization maps the linear trend in vegetation cover across the globe. For each 2°×2° grid cell, we computed the per-year linear regression slope from July-only NDVI measurements. Each dot's color shows whether vegetation has increased (green), decreased (brown), or stayed stable (tan) over two decades.

Click any dot to see its full 24-year time series. Shift+drag to select a region and view summary statistics. Use the region selector to zoom into areas of interest.

Design rationale

Why this visualization works

This visualization focuses on long-term vegetation change by mapping MODIS NDVI trends from 2001 to 2024. Instead of showing a single year or seasonal snapshot, the graph summarizes whether each location has become greener, browner, or relatively stable over time. This design choice supports the central question of whether the Earth is getting greener or browner, while still allowing users to investigate regional differences.

We chose a dot-grid map rather than a choropleth map because each dot represents the same type of spatial unit: a 2°×2° grid cell. This avoids giving too much visual weight to physically large land areas and makes each sampled location easier to compare. A choropleth map could make large regions such as Siberia or the Sahara dominate the viewer’s attention, even if the strongest trend signals occur elsewhere. The dot grid gives a cleaner overview of where vegetation trends are concentrated.

Color encodes the direction and strength of the NDVI trend. Brown indicates browning or vegetation decline, green indicates greening or vegetation increase, and tan indicates relative stability. We used discrete color bins instead of a continuous gradient because the main task is classification: users should be able to quickly recognize which locations are strongly greening, mildly greening, stable, mildly browning, or strongly browning. The thresholded palette makes the map easier to interpret at a glance.

The main interactions are region selection, significance filtering, tooltips, point selection, and brushing. The region dropdown helps users quickly zoom into meaningful environmental case studies such as the Amazon, India, the Sahel, California, and the boreal region. The significance filter allows users to focus on points with stronger model fit, reducing visual noise when users want more confident trend patterns. Tooltips provide details-on-demand, while clicking a dot opens the full 24-year NDVI time series for that location.

The time-series panel is important because it connects the map-level trend to the underlying temporal data. Without it, the viewer would only see the final trend classification and might not understand whether the pattern is steady, noisy, or driven by a few extreme years. Showing the annual NDVI values and regression line lets users verify the trend behind each dot and compare local patterns against the broader map.

The regional highlights section adds guided interpretation without removing user freedom. The map supports open-ended exploration, but the highlights direct users toward meaningful environmental stories such as Amazon deforestation, agricultural intensification in India, and climate-driven greening in northern latitudes. This helps the visualization work both as an exploratory tool and as an explanatory climate-data story.

Development process

Our development process began with exploring NASA MODIS NDVI data and deciding how to transform a time-based satellite dataset into an interpretable interactive map. We focused on July-only NDVI values from 2001 to 2024 so that the trend calculation compared the same seasonal window across years. For each grid cell, we computed a linear regression slope to estimate whether vegetation increased, decreased, or remained stable over time.

The implementation was built in D3 using SVG marks, an equirectangular projection, and interactive event handling. One part of the team focused on the data pipeline and trend calculation, while another part focused on the interface, map rendering, color scale, and interaction design. The most time-consuming parts were converting the NDVI data into a format that could load efficiently in the browser, choosing thresholds that made the trend categories readable, and debugging interactions such as click-to-select, brushing, and region filtering.

We also iterated on the visual design after testing early versions. A continuous color ramp made subtle trends difficult to classify, so we switched to five discrete bins. We also kept the regional highlights section because it helped users connect the abstract NDVI trend values to real environmental narratives. The final version balances overview, filtering, details-on-demand, and guided storytelling.
